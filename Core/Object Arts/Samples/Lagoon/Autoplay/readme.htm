All of the installations on the CD make use of the Microsoft Windows Installer mechanism. This has a number of advantages compared with earlier setup technology which means that installations should be easier and more robust. To make use of the MSI you must have an appropriate version of the Windows Installer (1.1) available on your machine before attempting to install any of the downloads on this page. Note that your operating system may already have MSI available, either because it came as part of the OS or because another application has installed it. If you are not sure, and want to save on additional download time, try the Dolphin product links first to see if the MSI file type is recognized by your system and can be opened as an installation file. Dolphin Smalltalk Professional

Dolphin Smalltalk is a fully featured rapid software development system for PC based Microsoft Windows™ applications. It is a highly productive development environment targeted at the graphical Visual Basic / C++ / Java developer who wishes to adopt a pure approach to object-oriented programming. Dolphin Smalltalk takes advantage of the latest Microsoft operating system features and enables the full functionality of the Microsoft Win32 APIs to be accessed from within the Smalltalk environment. As a result, applications built using Dolphin are fully featured, yet fast to market and easy to maintain. Dolphin Smalltalk works with Windows™ 95, 98, ME, NT4 or 2000.

Features

  • Dolphin applications can access all the underlying features of the Windows operating system including COM, XML and DirectX.
  • Small memory footprint of deployed applications compared to other Smalltalk development tools. 
  • Full ActiveX Control and Automation Support.
  • Unique Model-View Presenter application development framework.
  • Ability to deploy applications as web applets for execution within a web browser.
  • Includes a Live Update facility to ensure that the development environment can easily be kept up to date with the latest enhancements and bug fixes.

Dolphin Smalltalk Professional+Source Tracking System

This is version of Dolphin Smalltalk Professional that has been pre-loaded with STS (Source Tracking System)- an ENVY™-like clone specially written for the Dolphin environment.

STS uses an object oriented database, OmniBase, as a source code repository where all changes to methods, classes or packages are stored as soon as they are made. Features include browsing of old method/class/package/project/view editions, comparing changes between them, loading partial code from packages without installing them, merging code from different package editions and team development support. Source Tracking System also introduces projects into the Dolphin Smalltalk environment with which you can group multiple package versions together and load them all at once in a predefined order.

STS is available as an add-on to Dolphin from http://www.gorisek.com. This trial edition is time-limited to expire on 1st June 2001. To use the product after this point you must purchase a full Dolphin serial number from http://www.object-arts.com and a copy of STS from the preceding web site.
